# Brief

We are asked to add the one small visual icon like dot(coloured or black) or badge right side of the google calender month or week view. This indicates the user that this day contains at least one event I’ve marked as non-negotiable.

# Assumptions

- All non negotiable events have equal priority.
- This indicator is not interactive only visual.
- User will be able to define the event non-negotiable.
- Only one indicator for one or more than one non-negotiable event.
